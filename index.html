Problem:

Solve the following arithmetic question: $56 + 45 - 909 \times 81$

Step-by-Step Solution:

  1. Following **PEMDAS/BODMAS**, multiplication is performed first.
  2. $909 \times 81 = 73629$
  3. Now the equation is: $56 + 45 - 73629$
  4. $56 + 45 = 101$
  5. $101 - 73629 = -73528$

Python Code:

# Python correctly follows the order of operations
result = 56 + 45 - 909 * 81
print(f"Result: {result}")

Output:

Result: -73528

Problem:

Solve the following arithmetic question: $(50.50)^2 + 2^3$

Step-by-Step Solution:

  5. $2550.25 + 8 = 2558.25$

Result: 2558.25

Step-by-Step Solution:

  1. These operations require Python's built-in `math` module.
  2. **Important:** Python's trigonometric functions (`sin`, `cos`) expect angles in **radians**, not degrees.
  3. We must first convert degrees to radians using `math.radians(degrees)`.
  4. $\sin(90^{\circ})$: `math.sin(math.radians(90))` $\rightarrow$ $1.0$
  5. $\cos(45^{\circ})$: `math.cos(math.radians(45))` $\rightarrow$ $\approx 0.7071$
  6. $\sin(45^{\circ}) + \cos(30^{\circ})$: `math.sin(math.radians(45)) + math.cos(math.radians(30))` $\rightarrow$ $\approx 0.7071 + 0.8660 = 1.5731$
  7. $\log_{10}(50)$: Use `math.log10(50)` $\rightarrow$ $\approx 1.699$

Problem:

Write a function to find area of Trapezoid: $height=10cm$, $base1=20cm$, $base2=15cm$.

Step-by-Step Solution:

  1. The formula for the area of a trapezoid is: $Area = \frac{base1 + base2}{2} \times height$.
  2. Define a function `find_trapezoid_area` that accepts `height`, `base1`, and `base2` as arguments.
  3. Inside the function, implement the formula and return the result.
  4. Call the function with the given values: $Area = \frac{20 + 15}{2} \times 10 = \frac{35}{2} \times 10 = 17.5 \times 10 = 175.0$.

Python Code:

def find_trapezoid_area(height, base1, base2):
    """Calculates the area of a trapezoid."""
    area = ((base1 + base2) / 2) * height
    return area

# Given values
h = 10
b1 = 20
b2 = 15

# Calculate and print the area
area = find_trapezoid_area(h, b1, b2)
print(f"The area of the trapezoid is: {area} cm²")

The area of the trapezoid is: 175.0 cm²

Problem:

Write a function to find Volume of Cone: $height=0.5m$, $radius=21.45cm$.

Step-by-Step Solution:

  1. The formula for the volume of a cone is: $V = \frac{1}{3} \pi r^2 h$.
  2. **Crucial:** The units are mixed (meters and centimeters). We must convert them to be consistent. Let's convert everything to meters.
  3. $height = 0.5m$ (already in meters)
  4. $radius = 21.45cm = 0.2145m$ (divide by 100)
  5. We need `math.pi` for $\pi$.
  6. $V = \frac{1}{3} \times \pi \times (0.2145)^2 \times 0.5 \approx 0.02419 m^3$.

Python Code:

import math

def find_cone_volume(height_m, radius_cm):
    """Calculates the volume of a cone, converting cm to m."""
    
    # Convert radius from cm to meters
    radius_m = radius_cm / 100
    
    # Calculate volume
    volume = (1/3) * math.pi * (radius_m ** 2) * height_m
    return volume

# Given values
h = 0.5
r = 21.45

# Calculate and print the volume
volume = find_cone_volume(h, r)
print(f"The volume of the cone is: {volume:.5f} m³")

Output:

The volume of the cone is: 0.02419 m³

Note:

This function replaces elements that are *exactly* "A" or "a". It does not replace substrings, so "AA" and "aa" will be unchanged.

def replace_A_with_XX(input_list):
    """
    Replaces all list elements that are exactly "A" or "a" with "XX".
    Iterates in-place and returns the modified list.
    """
    # We iterate over the *indices* so we can modify the list
    for i in range(len(input_list)):
        if input_list[i] == "A" or input_list[i] == "a":
            input_list[i] = "XX"
    return input_list

# Given List

def get_ordinal_suffix(day):
    """Returns the ordinal suffix (st, nd, rd, th) for a given day."""
    if 11 <= day <= 13:
        return 'th'
    suffixes = {1: 'st', 2: 'nd', 3: 'rd'}
    return suffixes.get(day % 10, 'th')

import numpy as np

def get_first_n_primes(n):
    """Generates a list of the first n prime numbers."""
    primes = []
    num = 2  # Start checking from 2
    while len(primes) < n:
        is_prime = True
        # Check for factors from 2 up to the square root of num
        for i in range(2, int(num**0.5) + 1):
            if num % i == 0:
                is_prime = False
                break
        
        if is_prime:
            primes.append(num)
        
        num += 1 # Move to the next number
    return primes

# 1. Get the list of the first 50 primes
